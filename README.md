# Igbe Laara News

News website for Igbe Laara, Lagos, Nigeria

## Structure
- `/frontend` - React + Vite frontend
- `/backend` - Express + MongoDB backend

## Local Development

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```

## Deployment
Deploy frontend and backend separately to Vercel
